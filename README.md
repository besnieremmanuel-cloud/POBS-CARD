# POBS-CARD
free calculator for the preoperative risk evaluation of moderate-to-severe bleeding in cardiac surgery.
This calculator has been elaborated from the publication of Besnier et al in the JTCVS Open. [DOI: ](https://doi.org/10.1016/j.xjon.2024.04.008).
It is intended for research only as the score has not been properly validated in external study, appart from the original publication
